DECIMAL:

It is a number system wherein it uses digits from 0 to 9 in order to represent a number that is base 10.

Example: 10 in Decimal is represented as 1010 in Binary, 12 in Octal, and A in Hexadecimal.

Common Uses: Everyday calculations, finance, general use.

Fun Fact: The Decimal system is based on 10 because humans typically have 10 fingers.

BINARY:

It is a number system wherein it uses base-2 numbers that primarily uses only digits 1's and 0's.

Example: 10 in Decimal is represented as 1010 in Binary, 12 in Octal, and A in Hexadecimal.

Common Uses: Digital electronics, computer programming, data representation.

Fun Fact: Binary is the foundation of all modern computing systems.

OCTAL:

It is a number system wherein it uses base-8 numbers that primarily uses only digits from 0 to 7.

Example: 10 in Decimal is represented as 12 in Octal, 1010 in Binary, and A in Hexadecimal.

Common Uses: Older computer systems, Unix file permissions.

Fun Fact: Some cultures, such as the Yuki language in California and the Pamean languages in Mexico, also use octal numbering systems. This is because they use the spaces between their fingers instead of counting their fingers.

HEXADECIMAL:

It is a number system wherein it uses base-16 numbers which uses digits from 0 to 9 and letters A to F.

Example: 10 in Decimal is represented as A in Hexadecimal, 1010 in Binary, and 12 in Octal.

Common Uses: Memory addresses, color codes in web design, system programming.

Fun Fact: One hexadecimal digit represents four binary digits, also known as a nibble.
